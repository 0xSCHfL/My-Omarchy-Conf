# My Omarchy Dotfiles

A collection of my personal configuration files for Bash, Starship, and other tools to make my terminal environment consistent and productive.

## Features

- **Bash**: Custom `.bashrc` with aliases, functions, and environment variables.
- **Starship**: Custom prompt configuration with language versions, Git status, and Docker context.
- **Fonts**: Supports Nerd Fonts for beautiful Powerline-style prompts.
- **Tools**: Includes support for `fzf`, `zoxide`, and other productivity utilities.

## Installation

Clone the repository:

```bash
git clone git@github.com:0xSCHfL/My-Omarchy-Conf.git ~/.local/share/mybash
```
Change The directoy
```bash
cd mybash
```
Run the Install script
```bash
./mybash-setup.sh
```
